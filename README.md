# 북구 · 대구광역시

> 북구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-daegu-buk.github.io/**

대구광역시 북구는 금호강이 도심을 크게 감아 도는 강북의 자치구로, 조선 전기 학자 서거정이 대구십영에서 노래한 침산(砧山)을 품은 고을입니다. 침산동 옥산로에 자리한 북구청을 중심으로 산격동의 대구광역시 산격청사, 경북대학교, 엑스코(EXCO), 대구오페라하우스, 대구종합유통단지가 밀집해 대구의 행정·전시·유통 기능이 집약된 지역입니다. 옛 칠성시장·매천도매시장의 상권과 제3산업단지·검단산업단지의 제조 기반 위에, 강북(칠곡)·태전·동천의 신흥 주거단지가 더해져 43만 명 안팎이 살아가는 대구 인구 상위권 자치구로 성장했습니다. 함지산 능선의 구암동 고분군과 팔거산성이 팔거평야를 다스리던 삼국시대 세력의 존재를 증언하고, 노곡동 금호강 하중도가 도심 속 유채·코스모스 명소로 자리 잡아 역사와 산업과 자연이 겹쳐 있는 곳입니다.
